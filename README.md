## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/ashacv17/asha-portfolio.git
```

Navigate to the project folder:
```bash
cd asha-portfolio
```

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev

```


The application will run at:
```bash
http://localhost:5173

```
